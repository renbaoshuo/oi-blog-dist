题面

题目描述

输入格式

输入共 222 行：

  • 第 111 行包含 111 个正整数 N，表示 NNN 辆列车。
  • 第 222 行包含 NNN 个正整数，为 111 至 NNN 的一个排列，表示进站次序。

输出格式

输出共一行，包含一个整数，表示站台内轨道数 KKK 的最小值。

输入样例 1

3
1 2 3

输出样例 1

3

输入样例 2

9
1 3 2 4 8 6 9 5 7

输出样例 2

5

数据规模与约定

  • 对于 30% 的数据，N≤10N ≤ 10N≤10;
  • 对于 70% 的数据，N≤2000N ≤ 2000N≤2000;
  • 对于 100% 的数据，N≤100000N ≤ 100000N≤100000;

思路

贪心解决。

容易想到最优方案是让列车进入队尾车辆编号刚好大于要入队的车的编号的队列，使用 set 维护队尾即可。

代码

#include <bits/stdc++.h>

using namespace std;

int n, l, x, ans;
set<int> s;

int main() {
    cin >> n;
    for (int i = 1; i <= n; i++) {
        cin >> x;
        auto it = s.upper_bound(x);
        if (it == s.end()) {
            ans++;
        } else {
            s.erase(*it);
        }
        s.insert(x);
    }
    cout << ans << endl;
    return 0;
}
