#题面

#题目描述

小奇总是在数学课上思考奇怪的问题。

给定一个长度为 nnn 的数列，以及 mmm 次询问，每次给出三个数 lll，rrr 和 ppp，询问 (al′+al′+1+⋯+ar′)(modp)(a_{l'} + a_{l' + 1} + \dots + a_{r'}) \pmod p(al′​+al′+1​+⋯+ar′​)(modp) 的最小值（其中 l≤l′≤r′≤rl \leq l' \leq r' \leq rl≤l′≤r′≤r）。

即模意义下的区间子串和最小值。

#输入格式

第一行包含两个正整数 nnn 和 mmm，表示数列的长度和询问的个数。

第二行包含 nnn 个整数 a1,…,ana_1, \dots, a_na1​,…,an​。

接下来 mmm 行，每行三个数 lll，rrr 和 ppp，代表一次询问。

#输出格式

对于每次询问，输出一行一个整数表示要求的结果。

#输入输出样例

输入样例 #1

4 2
8 15 9 9
1 3 10
1 4 17

输出样例 #1

2
1

#思路

20 分

对于每个询问，在区间内暴力枚举子串求和、取模即可。复杂度为 O(mn3)O(mn^3)O(mn3)。

40 分

考虑在 20 分做法的基础上优化。不难发现，区间求和可以使用前缀和优化。复杂度为 O(mn2)O(mn^2)O(mn2)。

70 分

通过 抽屉原理 可知，当询问的区间长度大于等于 ppp 时（即 r−l≥pr - l \geq pr−l≥p 时），一定有两个前缀和相减等于 000，此时直接输出 000 即可。

再考虑枚举 rrr，可知当 kkk 是 sumr\mathit{sum}_{r}sumr​ 的前驱时 sumr−k\mathit{sum}_{r} - ksumr​−k 最小。时间复杂度为 O(mp2)O(mp^2)O(mp2)。

100 分

使用 std::set 维护前驱即可。时间复杂度为 O(mplog⁡p)O(mp \log p)O(mplogp)。

#代码

40 分 
#include <iostream>

using std::cin;
using std::cout;
using std::endl;

const int N = 100005;

int n, m, l, r, p, a[N];

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }
    while (m--) {
        cin >> l >> r >> p;
        int ans = p;
        for (int i = l; i <= r; i++) {
            int sum = 0;
            for (int j = i; j <= r; j++) {
                sum = (sum + a[j]) % p;
                ans = std::min(ans, sum);
            }
        }
        cout << ans << endl;
    }
    return 0;
}

        for (int i = l; i <= r; i++) {
            sum[i] = (sum[i - 1] + a[i]) % p;
            for (int j = sum[i]; j >= 0; j--) {
                if (vis[j]) {

#include <iostream>
#include <set>

using std::cin;
using std::cout;
using std::endl;

const int N = 500005;

int n, m, l, r, p, a[N];
long long sum[N];

int main() {
    cin >> n >> m;
    for (int i = 1; i <= n; i++) {
        cin >> a[i];
    }
    while (m--) {
        long long ans = 0x3f3f3f3f;
        cin >> l >> r >> p;
        if (r - l >= p) {
            cout << 0 << endl;
            continue;
        }
        std::set<long long> s;
        s.insert(0);
        sum[l - 1] = 0;
        for (int i = l; i <= r; i++) {
            sum[i] = (sum[i - 1] + a[i]) % p;
            ans = std::min(ans, sum[i] - *(--s.upper_bound(sum[i])));
            s.insert(sum[i]);
        }
        cout << ans << endl;
    }
    return 0;
}
