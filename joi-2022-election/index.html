题面

题目描述

JOI 共和国由 NNN 个州组成，现在 JOI 共和国正在进行选举。

理恵想要赢得选举，她决定以发表演讲的方式来提高自己的支持率，具体的，她可以发表任意时间长度的演讲，这些演讲会造成一定的影响：

  • 如果在这个州的演讲时间达到 AiA_iAi​，那么她将会赢得一张选票；
  • 如果在这个州的演讲时间达到 BiB_iBi​，那么她将会赢得一名协作者，协作者可以与她同时在任何州发表演讲，如果多人同时在一个州发表演讲，计入多倍演讲时间。
  • 有可能无法获得协作者，此时 Bi=−1B_i=-1Bi​=−1，此外我们保证 Bi≥AiB_i\ge A_iBi​≥Ai​。

理恵想尽快得到 KKK 张选票，请计算最小的演讲时间。

输入格式

第一行一个整数 NNN。

第二行一个整数 KKK。

接下来 NNN 行，一行两个整数 Ai,BiA_i,B_iAi​,Bi​。

输出格式

输出一个实数，表示需要的最小演讲时间，你的答案与标准答案的绝对误差不应超过 0.010.010.01。

思路

原题面可以转化为：

给出一个长度为 nnn 的序列，从中选出 kkk 个元素，设 c=1c = 1c=1，对于这 kkk 个元素，在排序后有以下操作：

  1. 耗费 aic\frac{a_i}{c}cai​​ 的时间处理这个元素；
  2. 耗费 bic\frac{b_i}{c}cbi​​ 的时间处理这个元素并使 ccc 增加 111。

求处理完全部 kkk 个元素的最小花费。

显然先进行操作 2 比先进行操作 1 要优，并且优先选择 bib_ibi​ 小的比其他方式优。

然后考虑 DP。设 fi,jf_{i, j}fi,j​ 表示在前 iii 个州中招收协作者的最小耗时，其中 iii 表示当前所处的州的编号，jjj 表示到达当前状态获得了多少协作者。再预处理出 gi,jg_{i, j}gi,j​ 表示在 ai∼na_{i \sim n}ai∼n​ 中前 jjj 小的数的总和。最后将 fff 和 ggg 中对应的值相加即可。

详细实现可以查看下方代码。

代码

#include <iostream>
#include <algorithm>
#include <cstring>
#include <iomanip>

using std::cin;
using std::cout;
const char endl = '\n';

const int N = 505;

int n, k;
std::pair<int, unsigned int> p[N], q[N];
double f[N][N], g[N][N], ans = 1e9;

int main() {
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n >> k;

    for (int i = 1; i <= n; i++) {
        cin >> p[i].first >> p[i].second;
    }

    std::sort(p + 1, p + 1 + n, [&](auto a, auto b) -> bool {
        return a.second < b.second;
    });

    for (int i = 1; i <= n; i++) {
        q[i] = p[i];
    }

    // 预处理 i ~ n 中 a_i 的前 j 小之和
    for (int i = n; i; i--) {
        std::sort(q + i, q + 1 + n, [&](auto a, auto b) -> bool {
            return a.first < b.first;
        });

        for (int j = i; j <= n; j++) {
            g[i][j - i + 1] = g[i][j - i] + q[j].first;
        }
    }

    for (int t = 0; t < k; t++) {  // 枚举招纳协作者的数量
        for (int i = 0; i <= k; i++) {
            for (int j = 0; j <= t; j++) {
                f[i][j] = 1e9;
            }
        }

        f[0][0] = 0;  // 演讲前耗时为 0

        for (int i = 1; i <= k; i++) {
            // 还未招协作者的时候直接从上一次转移
            f[i][0] = f[i - 1][0] + static_cast<double>(p[i].first) / (t + 1);  // 还要加上 Rie 本人

            for (int j = 1; j <= t; j++) {
                f[i][j] = std::min(
                    f[i - 1][j - 1] + static_cast<double>(p[i].second) / j,    // 需要招协作者的立即演讲
                    f[i - 1][j] + static_cast<double>(p[i].first) / (t + 1));  // 不招协作者的在招满后再演讲
            }
        }

        for (int i = 0; i <= k; i++) {
            ans = std::min(ans, f[i][t] + g[i + 1][k - i] / (t + 1));  // 增加协作者耗时 + 纯演讲耗时
        }
    }

    cout << std::fixed << std::setprecision(15) << ans << endl;

    return 0;
}
