对于任意整数 a,ba, ba,b，存在一对整数 x,yx, yx,y 满足 ax+by=gcd(a,b)ax + by = gcd(a, b)ax+by=gcd(a,b)。

扩展欧几里得算法

扩展欧几里得算法是用来求二元一次不定方程（如 (1)(1)(1) 式）的一组特解的算法。

ax+by=c(1)\tag{1} ax + by = cax+by=c(1)

(1)(1)(1) 式有解当且仅当 gcd⁡(a,b) ∣ c\gcd(a, b) ~|~ cgcd(a,b) ∣ c。

实现

对于每次递归，

  • 当 b=0b = 0b=0 时，
    ax+by=aax + by = aax+by=a，故 x=1,y=0x = 1, y = 0x=1,y=0。
  • 当 b≠0b \ne 0b=0 时，
    x=x,y=y−⌊ab⌋xx = x, y = y - \lfloor\frac{a}{b} \rfloor xx=x,y=y−⌊ba​⌋x。

证明：

设 d=gcd⁡(a,b)d = \gcd(a, b)d=gcd(a,b)，由欧几里得算法得：

d=ax+by=by+(a mod b)x=by+(a−⌊ab⌋b)x=ax+b(y−⌊ab⌋x)\begin{aligned} d & = ax + by \\ & = by + (a \bmod b)x \\ & = by + (a - \lfloor \frac{a}{b} \rfloor b )x \\ & = ax + b(y - \lfloor \frac{a}{b} \rfloor x) \end{aligned}d​=ax+by=by+(amodb)x=by+(a−⌊ba​⌋b)x=ax+b(y−⌊ba​⌋x)​

则上述 b≠0b \ne 0b=0 情况中的式子成立。

证毕。

代码

int exgcd(int a, int b, int& x, int& y) {
    if (!b) {
        x = 1, y = 0;
        return a;
    }

    int g = exgcd(b, a % b, y, x);
    y -= a / b * x;
    return g;
}

逆元

若 ax≡1 ( mod  b)ax \equiv 1 ~(\bmod~ b)ax≡1 (mod b)，且 a,ba, ba,b 互质，则称 xxx 为 aaa 的逆元，记为 a−1a^{-1}a−1。

可将上式转化为：

ax+by=1(2)\tag{2} ax + by = 1ax+by=1(2)

这样可以就使用扩展欧几里得算法求出 (2)(2)(2) 式的解，xxx 的最小解即为 aaa 的逆元。
