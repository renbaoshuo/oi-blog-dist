#排列数

#定义

从 nnn 个不同的元素中取出 mmm 个不同的元素作为一个排列，产生的不同排列数量记为 Anm\mathrm A_{n}^{m}Anm​。排列数的计算公式为：

Anm=n(n−1)(n−2)⋯(n−m+1)=n!(n−m)!\mathrm A_{n}^{m} = n(n - 1)(n - 2) \cdots (n - m + 1) = \frac{n!}{(n - m)!} Anm​=n(n−1)(n−2)⋯(n−m+1)=(n−m)!n!​

#特化

  1. 当 m=0m = 0m=0 时，只有一种方案即什么都不选；
  2. 当 m=nm = nm=n 时，排列为全排列 Anm=Ann=n!\mathrm A_{n}^{m} = \mathrm A_{n}^{n} = n!Anm​=Ann​=n!；
  3. 当 m>nm > nm>n 时，排列数为 000。

#组合数

#定义

从 nnn 个不同元素种取出 mmm 个元素作为一个集合，产生的不同集合数量记为 Cnm\mathrm C_{n}^{m}Cnm​，也常用 (nm)\displaystyle \binom{n}{m}(mn​) 表示。组合数的计算公式为：

Cnm=AnmAmm=n!m!(n−m)!\mathrm C_{n}^{m} = \frac{\mathrm A_{n}^{m}}{\mathrm A_{m}^{m}} = \frac{n!}{m!(n - m)!} Cnm​=Amm​Anm​​=m!(n−m)!n!​

#特化

  1. 当 m=0m = 0m=0 时，只有一种方案即什么都不选；
  2. 当 m=nm = nm=n 时，只有一种方案即全选；
  3. 当 m>nm > nm>n 时，组合数为 000。

#性质

  1. (nm)=(nn−m)\displaystyle \binom{n}{m} = \binom{n}{n - m}(mn​)=(n−mn​)，选择 mmm 个相当于选择 n−mn - mn−m 个留下来；
  2. Pascal 公式：(nm)=(n−1m)+(n−1m−1)\displaystyle \binom{n}{m} = \binom{n - 1}{m} + \binom{n - 1}{m - 1}(mn​)=(mn−1​)+(m−1n−1​)，常用于递推求组合数；
  3. ∑m=0n(nm)=2n\displaystyle \sum_{m = 0}^{n} \binom{n}{m} = 2^nm=0∑n​(mn​)=2n；
  4. ∑k=0n(−1)k(ni)=[n=0]\displaystyle \sum_{k = 0}^{n} (-1)^k \binom{n}{i} = [n = 0]k=0∑n​(−1)k(in​)=[n=0]；

#Catalan 数（卡特兰数）

#定义

Catn={1(n∈{1,2})∑i=1nCati−1⋅Catn−i(n≥2,n∈N∗)\mathit{Cat}_n = \begin{cases} 1 & (n \in \{1, 2\}) \\ \sum_{i = 1}^{n} \mathit{Cat}_{i - 1} \cdot \mathit{Cat}_{n - i} & (n \geq 2, n \in \N^*) \\ \end{cases} Catn​={1∑i=1n​Cati−1​⋅Catn−i​​(n∈{1,2})(n≥2,n∈N∗)​

递推定义：

Catn=4n−2n+1Catn−1\mathit{Cat}_n = \frac{4n - 2}{n + 1} \mathit{Cat}_{n - 1} Catn​=n+14n−2​Catn−1​

通项公式：

Catn=(2nn)n+1 (n≥2,n∈N∗)\mathit{Cat}_n = \frac{\binom{2n}{n}}{n + 1} ~~~ (n \geq 2, n \in \N^*) Catn​=n+1(n2n​)​ (n≥2,n∈N∗)

#组合意义

  1. 给定 nnn 个 000 和 nnn 个 111，他们按照某种顺序排成一个长度为 2n2n2n 的序列，满足任意前缀中 000 的个数都不少于 111 的个数的序列的数量可以记作 Catn\mathit{Cat}_nCatn​；
  2. 一个栈的进栈序列为 1,2,3,…,n1, 2, 3, \dots, n1,2,3,…,n，不同的出栈序列的方案数；
  3. nnn 个节点可以构造的二叉树数目；
  4. 在圆上选择 2n2n2n 个点，将这些点成对连接起来使得所得到的 nnn 条线段不相交的方法数；
  5. 等等。
