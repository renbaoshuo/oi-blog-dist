题面

题目描述

小 X 遇到了一道题：

给定自然数 a,ba, ba,b，求满足下列条件的自然数对 (x,y)(x, y)(x,y) 的个数：

y2−x2=ax+by^2 - x^2 = ax + by2−x2=ax+b

他不会，只好求助于精通数学的你。

如果有无限多个自然数对满足条件，那么你只需要输出 inf 即可。

输入格式

一行两个整数 a,ba, ba,b。

输出格式

如果个数有限，一行一个整数，表示个数。

如果个数无限，一行一个字符串 inf。

输入输出样例

样例输入 #1

5 15

样例输出 #1

1

样例解释 #1

(x,y)=(6,9)(x, y) = (6, 9)(x,y)=(6,9).

样例输入 #2

样例输出 #2

inf

样例输入 #3

样例输出 #3

0

样例输入 #4

样例输出 #4

样例输入 #5

样例输出 #5

思路

先观察题目中给出的这个式子：

y2−x2=ax+b(1)\tag{1} y^2 - x^2 = ax + by2−x2=ax+b(1)

对 (1)(1)(1) 式进行移项，得：

y2−b=x2+2ax(2)\tag{2} y^2 - b = x^2 + 2axy2−b=x2+2ax(2)

对 (2)(2)(2) 式右侧配方，得：

y2−b=(x+a2)2−a24(3)\tag{3} y^2 - b = (x + \frac{a}{2})^2 - \frac{a^2}{4}y2−b=(x+2a​)2−4a2​(3)

将 (3)(3)(3) 式两侧同乘 444，得：

4y2−4b=(2x+a)2−a2(4)\tag{4} 4y^2 - 4b = (2x + a)^2 - a^24y2−4b=(2x+a)2−a2(4)

对 (4)(4)(4) 式再次移项，得：

a2−4b=(2x+a)2−4y2(5)\tag{5} a^2 - 4b = (2x + a)^2 - 4y^2a2−4b=(2x+a)2−4y2(5)

展开 (5)(5)(5) 式左侧，得：

a2−4b=(2x+a+2y)(2x+a−2y)(6)\tag{6} a^2 - 4b = (2x + a + 2y) (2x + a - 2y)a2−4b=(2x+a+2y)(2x+a−2y)(6)

由题，显然 2x+a+2y>02x + a + 2y > 02x+a+2y>0，接下来分类讨论：

  1. 当 a2−4b<0a^2 - 4b < 0a2−4b<0 时，2x+a−2y<02x + a - 2y < 02x+a−2y<0：

    左右同乘 −1-1−1，得：

    4b−a2=(2y+2x+a)(2y−2x−a)(7)\tag{7} 4b - a^2 = (2y + 2x + a) (2y - 2x - a)4b−a2=(2y+2x+a)(2y−2x−a)(7)

    记 p=2y+2x+ap = 2y + 2x + ap=2y+2x+a，q=2y−2x−aq = 2y - 2x - aq=2y−2x−a（0<q<p0 < q < p0<q<p），显然可以将 4b−a24b - a^24b−a2 分解为两数之积可以得到一组 p,qp, qp,q。又有：

    p+q=4yp−q=4x+2a⇒p−q−2a=4x\begin{aligned} p + q &= 4y \\ p - q &= 4x + 2a \Rarr p - q - 2a = 4x \\ \end{aligned}p+qp−q​=4y=4x+2a⇒p−q−2a=4x​

    可以枚举 qqq 再据此计算出 ppp，进而再计算出 x,yx, yx,y 的取值。

  2. 当 a2−4b>0a^2 - 4b > 0a2−4b>0 时，2x+a−2y>02x + a - 2y > 02x+a−2y>0：

    记 p=2x+a+2yp = 2x + a + 2yp=2x+a+2y，q=2x+a−2yq = 2x + a - 2yq=2x+a−2y（0<q<p0 < q < p0<q<p），显然可以将 a2−4ba^2 - 4ba2−4b 分解为两数之积可以得到一组 p,qp, qp,q。又有：

    p+q=4x+2a⇒p+q−2a=4xp−q=4y\begin{aligned} p + q &= 4x + 2a \Rarr p + q - 2a = 4x \\ p - q &= 4y \\ \end{aligned}p+qp−q​=4x+2a⇒p+q−2a=4x=4y​

    可以枚举 qqq 再据此计算出 ppp，进而再计算出 x,yx, yx,y 的取值。

  3. 当 a2−4b=0a^2 - 4b = 0a2−4b=0 时，2x+a−2y=02x + a - 2y = 02x+a−2y=0：

    由 a2−4b=0a^2 - 4b = 0a2−4b=0 可知，aaa 为偶数。可得 x+a2=yx + \frac{a}{2} = yx+2a​=y。

    显然对于任意一个 xxx 都有一个与其对应的 yyy 满足条件。此时有无穷多组解。

代码

#include <cmath>
#include <iostream>

using std::cin;
using std::cout;
const char endl = '\n';

long long a, b, d, m, ans;

int main() {
    std::ios::sync_with_stdio(false);

    cin >> a >> b;

    d = a * a - b * 4;

    if (d == 0) {
        cout << "inf" << endl;

        exit(0);
    } else if (d < 0) {
        d *= -1;
        m = std::sqrt(d);

        for (long long q = 1; q <= m; q++) {
            if (d % q == 0) {
                long long p = d / q;

                long long x = p - q - 2 * a,
                          y = p + q;

                if (x % 4 || y % 4) continue;

                x /= 4, y /= 4;

                if (x >= 0 && y >= x) ans++;
            }
        }
    } else {  // d > 0
        m = std::sqrt(d);

        for (long long q = 1; q <= m; q++) {
            if (d % q == 0) {
                long long p = d / q;

                long long x = p + q - 2 * a,
                          y = p - q;

                if (x % 4 || y % 4) continue;

                x /= 4, y /= 4;

                if (x >= 0 && y >= x) ans++;
            }
        }
    }

    cout << ans << endl;

    return 0;
}
