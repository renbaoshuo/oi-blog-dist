题面

题目描述

我们把仅由一个或多个相同的数位组成的数字叫作「无聊的数字」。我们把 ≤10000\leq 10000≤10000 的这种数按照以下规则排列：

  • 首先，将仅由 111 组成的 ≤10000\leq 10000≤10000 的「无聊的数字」按照升序排列：1,11,111,1111,…1, 11, 111, 1111, \dots1,11,111,1111,…。
  • 再将仅由 222 组成的 ≤10000\leq 10000≤10000 的「无聊的数字」按照升序排列：2,22,222,2222,…2, 22, 222, 2222, \dots2,22,222,2222,…。
  • 以此类推。

给出 ttt 次询问，每次询问给定一个无聊的数字 xxx，请求出当数列排到 xxx 时，xxx 以及前面所有的“无聊的数字”的位数之和。

输入输出样例

样例输入 #1

4
22
9999
1
777

样例输出 #1

13
90
1
66

数据范围与约定

对于 100%100\%100% 的数据，1≤t≤36,1≤x≤99991 \leq t \leq 36, 1 \leq x \leq 99991≤t≤36,1≤x≤9999。

思路

先手算一下，可以找出规律：

  • (num mod 10)−1(num \bmod 10) - 1(nummod10)−1 是结果的十位上的数字。
  • 数字长度为 iii 时，个位上的数字就是 1+2+3+...+i1 + 2 + 3 + ... + i1+2+3+...+i 。

可以得到如下代码：

int num, ans = 0, i = 1;
cin >> num;
ans = ((num % 10) - 1) * 10;
while (num) {
    ans += i++;
    num /= 10;
}
cout << ans << endl;
