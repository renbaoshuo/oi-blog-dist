Codeforces - 1433A Boring Apartments

2020-10-22题解328 字

思路

先手算一下，可以找出规律：

  • (num mod 10)−1(num \bmod 10) - 1(nummod10)−1 是结果的十位上的数字。
  • 数字长度为 iii 时，个位上的数字就是 1+2+3+...+i1 + 2 + 3 + ... + i1+2+3+...+i 。

可以得到如下代码：

int num, ans = 0, i = 1;
cin >> num;
ans = ((num % 10) - 1) * 10;
while (num) {
    ans += i++;
    num /= 10;
}
cout << ans << endl;
