#前置知识

#异或和

集合 SSS 的异或和为 xori=1∣S∣Si\text{xor}_{i = 1}^{|S|} S_ixori=1∣S∣​Si​（即 S1 xor S2 xor … xor S∣S∣S_1 \text{ xor } S_2 \text{ xor } \dots \text{ xor } S_{|S|}S1​ xor S2​ xor … xor S∣S∣​）。

#张成

设 T⊆ST \subseteq ST⊆S，所有这样的子集 TTT 异或和组成的集合称为 SSS 的张成，记为 span(S)\text{span}(S)span(S)。即在 SSS 中选出任意多个数，其异或和的所有可能的结果组成的集合。

#线性相关与线性无关

对于一个集合 SSS，如果存在一个元素 SiS_iSi​ 使得 SSS 在去除这个元素后得到的集合 S′S'S′ 满足 Si∈span(S′)S_i \in \text{span}(S')Si​∈span(S′)，即存在一个元素 SiS_iSi​ 可以通过集合中的若干个剩余元素异或得到，则称集合 SSS 线性相关。

相对地，如果不存在这样的元素 SiS_iSi​，则称集合 SSS 线性无关。

#线性基

#定义

我们称集合 BBB 是集合 SSS 的线性基，当且仅当：

  1. S⊆span(B)S \subseteq \text{span}(B)S⊆span(B)；
  2. BBB 是线性无关的。

集合 BBB 中元素的个数称为线性基的 长度。

线性基有以下几个性质：

  1. SSS 中的任意元素都可以 唯一 表示为 BBB 中若干个元素的异或和（对应条件 1）；
  2. BBB 是极小的满足线性基性质的集合，它的任何真子集都不可能是线性基（对应条件 2）；
  3. BBB 中没有异或和为 000 的子集（对应条件 2）；
  4. BBB 中每个元素的二进制最高位互不相同；
  5. 在 BBB 中选取若干个元素异或起来得到一个元素，用这个元素去替换 BBB 中的任意一个元素，得到的新集合 B′B'B′ 张成的空间不变。

#构造

对集合 SSS 的每个数 xxx 转为二进制，自高位向低位扫描，若 xxx 的第 kkk 位为 111，如果 aka_kak​ 不存在，那么令 ak=xa_k = xak​=x 并结束扫描，如果存在则令 x=x xor akx = x \text{ xor } a_kx=x xor ak​。

在 Menci 的《线性基学习笔记》中还给出了另一种线性基的构造方法，这种方法将原本的三角基消为了对角基：

设集合 SSS 中最大的数在二进制意义下有 LLL 位，我们使用一个 [0…L][0 \ldots L][0…L] 的数组 aaa 来储存线性基。

  • 逆序枚举 ttt 所有为 111 的二进制位 j=L→0j = L \rightarrow 0j=L→0，对于每个 jjj
    • 如果 aj≠0a_j \neq 0aj​=0，则令 t=txorajt = t \mathbin{\text{xor}} a_jt=txoraj​；
    • 如果 aj=0a_j = 0aj​=0，则：
      • 枚举 k∈[0,j)k \in [0, j)k∈[0,j)，如果 ttt 的第 kkk 位为 111，则令 t=txorakt = t \mathbin{\text{xor}} a_kt=txorak​；
      • 枚举 k∈(j,L]k \in (j, L]k∈(j,L]，如果 aka_kak​ 的第 jjj 位为 111，则令 ak=akxorta_k = a_k \mathbin{\text{xor}} tak​=ak​xort；

这种线性基的构造方法保证了一个特殊性质，对于每一个 iii，aia_iai​ 有以下两种可能：

  1. ai=0a_i = 0ai​=0，并且：
    • 只有 满足 j>ij > ij>i 的 aja_jaj​（即，位于 aia_iai​ 后面的所有 aja_jaj​）的第 iii 个二进制位 可能 为 111。
  2. ai≠0a_i \neq 0ai​=0，并且：
    • 整个 aaa 数组中 只有 aia_iai​ 的第 iii 个二进制位为 111；
    • aia_iai​ 更高的二进制位（>i> i>i 的二进制位）一定为 000；
    • aia_iai​ 更低的二进制位（<i< i<i 的二进制位）可能为 111。

                x ^= a[i];
            } else {

                    if (x & (1ull << k)) x ^= a[k];
                }

                for (int k = i + 1; k <= 50; k++) {

#应用

#检查某个数是否能被表示成集合中若干元素的异或和

首先构造出这个集合的线性基，然后检测这个数是否能被插入到线性基中即可。

inline bool check(unsigned long long x) {
    for (int i = 50; ~i; i--) {
        if ((x >> i) & 1) {
            if (a[i]) {
                x ^= a[i];
            } else {
                // 能被插入表示集合的张成中不包括 x
                return false;
            }
        }
    }

    return true;
}

#求最大子集异或和

从高位到低位考虑基，若异或后变大，则异或。

由于 a0∼i−1a_{0 \sim i - 1}a0∼i−1​ 都不含 iii 及以上的位，其最大贡献仅为 2i−12^i - 12i−1。有两种情况：

  1. 若此时 ans\it{ans}ans 的第 iii 位是 000，则异或上 aia_iai​ 至少带来 2i2^i2i 的收益，大于后面可能的所有收益。同时，(ans ^ a[i]) > ans 也必然成立。
  2. 若此时 ans\it{ans}ans 的第 iii 位是 111，至少在这一位有 2i2^i2i 的亏损，大于后面的位可能的所有收益。同时，(ans ^ a[i]) > ans 也必不成立。
for (int i = 50; ~i; i++) {
    if ((ans ^ a[i]) > ans) ans ^= a[i];
}

这种做法由于将三角基消为了对角基，所以第二种情况不会出现。

for (int i = 0; i <= 50; i++) {
    ans ^= a[i];
}

#第 k 小子集异或和

先将按照上文中的「方法 2」建立对角线性基。

将 kkk 二进制拆分，每一位的 000 / 111 对应异或时选 / 不选线性基存在的这一位。

证明：线性基中存在的位的 000 / 111 唯一确定了一个异或出的数，由于每个位只在一个基中为 111，这些位组成的二进制数的大小就可以代表异或出的数的大小。

题目：LibreOJ #114. k 大异或和

预处理出线性基中所有存在的位：

for (int i = 0; i <= 50; i++) {
    if (a[i]) p[cnt++] = a[i];
}

对于每次询问 kkk：

if (cnt != n) k--;  // 确保选择的是非空子集

if (k >= (1ull << cnt)) {  // 不同的异或和数量不足 k
    cout << -1 << endl;
} else {
    ans = 0;

    for (int i = 0; i < cnt; i++) {
        if (k & (1ull << i)) ans ^= p[i];
    }

    cout << ans << endl;
}
