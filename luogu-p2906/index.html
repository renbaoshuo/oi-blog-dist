题面

题目描述

了解奶牛的人都知道奶牛是如何组成「奶牛社区」的。他们观察了 Farmer John 的 NNN 头奶牛（编号为 1∼N1 \sim N1∼N），它们在 XXX 和 YYY 坐标范围为 111 的牧场上放牧，每头奶牛都在自己唯一的整数直线坐标上。

如果满足以下两个标准中的至少一个，则两头奶牛是邻居：

  1. 两只奶牛的曼哈顿距离不超过 CCC，即 ∣Xi−xi∣+∣Yi−yi∣≤C|X_i - x_i| + |Y_i - y_i| \leq C∣Xi​−xi​∣+∣Yi​−yi​∣≤C；
  2. 两只奶牛有共同的邻居。即存在一只奶牛 kkk，使 iii 与 kkk，jjj 与 kkk 均同属一个群。

给定奶牛的位置和距离 CCC，确定「奶牛社区」的数量和最大的「奶牛社区」中的奶牛数量。

思路

机房巨佬于队曾经教过我们一个小 trick：曼哈顿距离和切比雪夫距离之间的转化。

曼哈顿坐标系是通过将切比雪夫坐标系旋转 π4\frac{\pi}{4}4π​ 后再缩小到原来的一半得到的。那么可以通过将 (x,y)(x, y)(x,y) 变为 (x+y,x−y)(x + y, x - y)(x+y,x−y) 得到曼哈顿坐标系。

记题中第 iii 个点 (Xi,Yi)(X_i, Y_i)(Xi​,Yi​) 的切比雪夫坐标为 (xi,yi)(x_i, y_i)(xi​,yi​)，那么限制 111 可以改写成：

  • 两只奶牛的切比雪夫距离不超过 CCC，即 max⁡(∣x1−x2∣,∣y1−y2∣)≤C\max(|x_1 - x_2|, |y_1 - y_2|) \leq Cmax(∣x1​−x2​∣,∣y1​−y2​∣)≤C。

那么可以先以 xxx 轴坐标为第一关键字，yyy 轴坐标为第二关键字从小到大排序，并使用并查集合并同一群的奶牛。然后使用 set 维护 yyy 轴坐标的值，在插入每个点之前将 xxx 轴坐标不合法的删除，然后使用 lower_bound 找到 yiy_iyi​ 的前驱和后继，如果满足限制则将其与 yiy_iyi​ 合并。最后统计连通块个数和最大连通块大小即可。

代码

#include <iostream>
#include <algorithm>
#include <set>

using std::cin;
using std::cout;
const char endl = '\n';

const int N = 1e5 + 5;

int n, c, fa[N], cnt[N], ans, max;
std::pair<int, int> points[N];
std::set<std::pair<int, int>> set;

int find(int x) {
    return fa[x] == x ? x : fa[x] = find(fa[x]);
}

inline void merge(int x, int y) {
    fa[find(x)] = find(y);
}

int main() {
    std::ios::sync_with_stdio(false);
    cin.tie(nullptr);

    cin >> n >> c;

    for (int i = 1, x, y; i <= n; i++) {
        cin >> x >> y;

        points[i] = std::make_pair(x + y, x - y);
        fa[i] = i;
    }

    std::sort(points + 1, points + 1 + n);

    set.insert(std::make_pair(points[1].second, 1));
    for (int i = 2, l = 1; i <= n; i++) {
        while (points[i].first - points[l].first > c) {
            set.erase(std::make_pair(points[l].second, l));
            l++;
        }

        auto it = set.lower_bound(std::make_pair(points[i].second, 0));

        if (it != set.end() && it->first - points[i].second <= c) {
            merge(i, it->second);
        }

        if (it != set.begin() && points[i].second - (--it)->first <= c) {
            merge(i, it->second);
        }

        set.insert(std::make_pair(points[i].second, i));
    }

    for (int i = 1; i <= n; i++) {
        if (find(i) == i) ans++;

        max = std::max(max, ++cnt[find(i)]);
    }

    cout << ans << ' ' << max << endl;

    return 0;
}
