中国剩余定理

中国剩余定理（Chinese Remainder Theorem，CRT）可求解如下形式的一元线性同余方程组（其中 n1,n2,⋯ ,nkn_1, n_2, \cdots, n_kn1​,n2​,⋯,nk​ 两两互质）：

{x≡a1(modn1)x≡a2(modn2)…x≡ak(modnk)\begin{cases} \begin{array}{cl} x \equiv a_1 & \pmod{n_1} \\ x \equiv a_2 & \pmod{n_2} \\ \ldots & \\ x \equiv a_k & \pmod{n_k} \\ \end{array} \end{cases}⎩⎨⎧​x≡a1​x≡a2​…x≡ak​​(modn1​)(modn2​)(modnk​)​​

算法流程

  1. 计算所有模数的积 nnn；
  2. 对于第 iii 个方程：
    1. 计算 mi=nnim_i = \frac{n}{n_i}mi​=ni​n​；
    2. 计算 mim_imi​ 在模 nin_ini​ 意义下的 逆元 mi−1m_i^{-1}mi−1​；
    3. 第 iii 个方程的解 ci=mi×mi−1c_i = m_i \times m_i^{-1}ci​=mi​×mi−1​（不要对 nin_ini​ 取模）。
  3. 求出方程组的唯一解 x=∑i=1kaimici(modn)x = \sum_{i = 1}^k a_i m_i c_i \pmod nx=∑i=1k​ai​mi​ci​(modn)。

代码实现

long long CRT() {
    long long mod = 1, ans = 0;

    for (int i = 1; i <= n; i++) {
        mod *= n[i];
    }

    for (int i = 1; i <= n; i++) {
        long long m = mod / n[i], x, y;
        exgcd(m, n[i], x, y);
        ans = (ans + a[i] * m * x % mod) % mod;
    }

    return (ans % mod + mod) % mod;
}

证明

因为 mi=nnim_i = \frac{n}{n_i}mi​=ni​n​ 是除 mim_imi​ 之外所有模数的倍数，所以 ∀k≠i,aimici≡0(modmk)\forall k \ne i, a_i m_i c_i \equiv 0 \pmod{m_k}∀k=i,ai​mi​ci​≡0(modmk​)。又因为 aimici≡ai(modmi)a_i m_i c_i \equiv a_i \pmod{m_i}ai​mi​ci​≡ai​(modmi​)，所以代入 x=∑i=1naimicix = \sum^{n}_{i = 1} a_i m_i c_ix=∑i=1n​ai​mi​ci​，原方程组成立。

证毕。
